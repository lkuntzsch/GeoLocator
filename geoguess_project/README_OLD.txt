Prerequesites:
- Python 3
- Flask
- Flask-CORS

Used:
- Chart.js
- Leaflet.js


Open 2 cmd windows:


First cmd, run flask server:
cd path
venv\Scripts\activate
python app.py

(Running on http: ...)



Second cmd, run html:
cd path
venv\Scripts\activate
python -m http.server 8000

(Serving HTTP on :: port ...)



Browser:
http://localhost:8000/index.html
